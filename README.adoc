=== build and run

.build, run and test using gradle / docker
[source,bash]
----
./gradlew
java -jar build/libs/*.jar
bash build/libs/*.jar

./gradlew build composeUp
./gradlew composeDown
----

.build, run and test using maven / docker
[source,bash]
----
./mvnw
java -jar target/*.jar
bash target/*.jar

bash mvnw docker-compose:up -P docker
bash mvnw docker-compose:down -P docker
----
